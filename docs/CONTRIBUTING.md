## Getting Started

1.  **Fork** repositori ini.
2.  **Clone** ke lokal Anda.
3.  Buat **Branch Baru** untuk fitur atau perbaikan bug Anda (`git checkout -b feature/amazing-feature`).

## Development Guidelines

### Code Style
*   Gunakan **ESModules** (`import/export`) di seluruh proyek (Frontend & Backend).
*   Gunakan **PascalCase** untuk komponen React (`MyComponent.jsx`).
*   Gunakan **camelCase** untuk fungsi dan variabel (`myFunction`).
*   Selalu gunakan **Tailwind Utility Classes** daripada CSS kustom jika memungkinkan.

### Commit Messages
Kami mengikuti konvensi **Conventional Commits**:
*   `feat: add new dashboard chart`
*   `fix: resolve login cors issue`
*   `docs: update API reference`
*   `style: improve button hover state`

### Pull Requests
1.  Pastikan kode Anda berjalan tanpa error di lokal.
2.  Jelaskan perubahan Anda secara rinci di deskripsi PR.
3.  Lampirkan screenshot jika ada perubahan visual UI.
